## Instructions to run API
Go to folder backend
<br>
Run npm install
<br>
Run nodemon index.js
<br>


## Testing the API
Run npm test

## Instructions to run React App
Go to folder frontend
<br>
Run npm install
<br>
Run npm start
<br>
On your browser go to localhost:3000, the app will be displayed there
